## Run Locally

Clone The Project

```bash
  git clone https://github.com/IA3TheDev-il/Area-Calculator
```

Go To The Project Directory

```bash
  cd Area-Calculator
```

Install Dependencies

```bash
  pip3 install -r requirements.txt
```

Start The Program

```bash
  python3 main.py
```
